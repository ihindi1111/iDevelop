Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. While HTML structures a webpage, CSS styles it, controlling the appearance of HTML elements, including layout, colors, fonts, and spacing. If HTML is the skeleton, CSS is the "clothing" that makes a webpage visually appealing.

What CSS Does:

  • Styles HTML elements with color, fonts, and sizes.
  • Controls layout, such as aligning items or setting spacing.
  • Makes pages responsive and interactive.

CSS Elements Explained:

  • CSS allows us to grab specific parts of our html file.
  • When we give an item an id or class we can single it out in the CSS.
  • Once we single out the element we can give it style such as color!
  • .{name of class} {color: red;} is how we style a specific class (devider)
  • #{name of id} {color : blue ;} is how we style a specific id
  • @media is how we give a responsive design!

Why CSS?

CSS makes our html stand out. When you make a website you want to make it as friendly as possible and CSS helps with that! When you pair it with tailwind or react for example you can make extremely user friendly resources that run smoothly

Example of CSS Code:

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}
h1 {
  color: #4CAF50;
  text-align: center;
}
